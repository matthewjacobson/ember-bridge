# Ember Bridge — User Manual

Ember Bridge is a small desktop app that lets the **Ember** design editor in your
web browser send embroidery designs to your WiFi-connected embroidery machine.

This manual is also available inside the app, on the **Help** page.

## Contents

1. [Getting started](getting-started.md) — install, first launch, and how the
   app lives in the menu bar.
2. [Connecting Ember](connecting-ember.md) — pairing your browser with the
   bridge and managing allowed sites.
3. [Adding machines](adding-machines.md) — finding, adding, testing, and
   managing machines.
4. [Setting up a dongle](dongle-setup.md) — the EmberConnect out-of-box WiFi
   setup over USB.
5. [Sending a design](sending-designs.md) — choosing a machine, file formats,
   and the upload queue.
6. [Troubleshooting](troubleshooting.md) — fixes for the most common problems.
